Para hacer funcionar esta primera versión del programa, es necesario agregar un dataset de formato ["FECHA","VALOR"],
por ahora solo esta leyendo de una energía y los datos todavía no se muestran, pero se crea un csv llamado forecast.csv donde 
se puede observar el resultado.

Version 0.2 12-01-2023
Added a yaml file to store the configuration of the program and select the energy to forecast.
Now the data is saved in a csv file with the name of the energy selected.
Still is not showing the data in the program.
And bla bla bla, still can be improved.

Tutorial:
1- Install python 3.10 or 3.9
2- Install requirements.txt with pip install -r requirements.txt
3- Run the program with python TimeSerie_Builder.py Chile.yml
4- Wait for the program to finish
5- See the results in the csv file with the name of the energy selected.
6- Enjoy
